# cognitive-arch-generic

Generic scaffold of the Cognitive Architecture SDK.

## Quickstart

1. Copy this folder into your project (or use it as-is)
2. Fill in `PROJECT.md` with your project's identity
3. Follow `BOOTSTRAP.md` — Step 0: `python sdk/session_start.py --arch-root .`

## What's included

- `sdk/` — 45 Python tools (zero external dependencies*)
- `templates/` — generation templates for phases, blocks, ADRs, agents
- `protocols/` — 40+ operational protocols
- `commands/` — slash-command specifications
- `governance/` — ux-voice.md, ux-config.yaml, tools-registry.yaml

*except pytest for tests

## What's NOT included (generated at runtime)

- Phase/block content — generated as you work
- Health reports, dashboard, pattern analysis — generated by SDK tools
- Notifications, proposals — generated by governor/pattern-mining

## Tests

```bash
python -m pytest sdk/tests/ -q
```
